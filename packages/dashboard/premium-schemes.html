Midnight Emerald

  • Feels like: Stripe Dashboard, Vercel Analytics, Plaid Console
  • Psychology: Emerald = trust, growth, security (perfect for auth)
  • Use case: When you need color without being loud
  • Differentiator: Not blue (everyone uses blue), stands out
  • Pro: Memorable, distinctive, professional

Graphite

  • Feels like: Linear, Arc Browser, Apple Pro Apps
  • Psychology: Pure focus, no distractions, content-first
  • Use case: When data is the hero, not the UI
  • Differentiator: Extreme minimalism, confident simplicity
  • Pro: Timeless, never goes out of style

Real Brand References

Midnight Emerald: Stripe uses deep blues with subtle green success states. Vercel uses pure blacks with cyan accents. This combines the trust of Stripe with Vercel's modern aesthetic.

Graphite: Linear proved grayscale can be luxurious. Arc Browser showed white buttons can be premium. This takes that philosophy and adds hierarchy through pure tonal variation.
